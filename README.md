# Play Phaser

Playing with PhaserJS.

## Instructions

### Install

```bah
nvm use 18
npm install
```

### Run

```bash
npm run serve --game=setinstone
```

### Build

```bash
npm run build --game=setinstone
```

## Tips and Tricks

### Append Images

```bash
# Horizontally
convert +append input1.png input2.png output.png
# Vertically
convert -append input1.png input2.png output.png
```
